Keywords: Linear regression, Prediction, Machine learning, Preprocessing, GradientBoost

Price prediction has always been a challenging task as there are several factors involved that tend to cause variation in prices. There is similar case with car price prediction. It is a difficult task to decide either a used car actually justifies the posted price. Factors such as mileage, model, year influence the car prices. The paper aims to develop machine learning models which include linear regression, gradient boosting algorithm, and decision tree in order to predict used car prices based on its features. In order to obtain results, we made an implementation using Apache Spark, which is a big data tool. Among several machine learning libraries used such as pandas and others, we have also preferred Apache spark for implementation. The dataset of PakWheels is utilized which originally have 56186 instances and 16 attributes. A considerable amount of work has been done on this research domain; however, very little amount of work has been done using pyspark. Our proposed models predicted accuracy with linear regression of about 50% while Gradient boosting algorithm obtained 89% and 86% accuracy with decision tree regression.
