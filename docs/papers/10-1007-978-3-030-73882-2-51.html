Keywords: Knowledge bases, Topic directory, Linking web pages, Multiple knowledge sourcing, Simpson’s diversity index

Mapping of Knowledge sources is one of the preliminary strategies for making Semantic Search foundational. The Web is flooded with information but making it meaningful requires the integration of useful resources and attenuated semantics. In this paper, an approach for integrating knowledge from several heterogeneous sources such as the knowledge bases, crowdsourced knowledge, and traditional web directories using an integration of semantic similarity and Simpson’s Diversity Index has been proposed for the construction of hierarchical topic directories, and yield topic-specific Linked Open Data. The proposed MKSTD framework is a knowledge sourcing and integration scheme for Linked Open Data generation and further leveraging the web pages to the generated topics based on the index words and the content words of the web pages for facilizing the Semantic Search. The MKSTD is evaluated on the Bookshelf dataset from the Gutenberg Project categorizations and an average F-Measure of 95.98% has been achieved with an nDCG score of 0.91 and a very low FDR of 0.06 for leveraging the indexed web pages with the topic hierarchies.
