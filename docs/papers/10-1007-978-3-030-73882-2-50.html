Keywords: Knowledge base, Query language, Random forest classification, Synonymization

Recommendation System is a filtering technology used to compute an item’s priority or rating to the user. The recommender system is used to generate audio or video playlist for users in media streaming applications, recommend products in e-commerce platforms, recommend posts on social media platforms, etc. The web usage mining technique is incorporated to extract topical words from the user’s historical web content. The data extracted is synonymized using WordNet, and a standard knowledge base is created. Classification algorithms such as Random Forest is employed on the knowledge base to classify the standard dataset, and the Ant Colony Optimization has been encompassed to determine the shortest route through graphs. This knowledge is fed to machine learning algorithms to classify the standard dataset.
