The proliferation of MOOCs has led to an issue of heterogeneity of data schemas and a problem of interoperability between different MOOC providers. The goal of this research paper is to propose a unified data model that serves as a basis for the design and implementation of a Linked Data repository. In order to achieve this goal, on the one hand, we rely on semantic web technologies: Linked Data and Ontology. On the other hand, we enrich this model with metadata related to the pedagogical aspect of MOOCs using Bloom’s Taxonomy as a theoretical foundation.
