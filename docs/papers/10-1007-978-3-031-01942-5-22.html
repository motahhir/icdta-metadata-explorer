From Sketch to Simulation in Virtual Laboratory Using Deep Learning

Keywords: Sketch to code, Virtual laboratories, Simulated learning environments, User interface, Simulation, CNN

Virtual labs are simulated learning environment softwares that allow students to complete laboratory experiments online or offline to explore concepts and theories without stepping into a physical science lab. Developing such a software code is a long process and a hard task to achieve that requires a skilled expert developer. Human free-hand sketches have been studied in various fields including sketch recognition, synthesis, sketch-based image retrieval, etc. This paper will present a new sketch paradigm to generate a simulation from a Hand sketch. We propose a novel method that uses Convolutional Neural Networks (CNNs), Recurrent Neural Networks (RNN) combined with a Laboratory as a Service (LaaS) in order to transform a user-drawing sketch into a simulation code. Our approach is based on the use of both SSD (Single Shut Detector) and Mobile-Net as feature extractions for object identification in the sketch, called corresponding web services, which eventually regenerate a simulation software. Our method has been trained and evaluated using the “Quick, Draw” dataset. Experimental results have demonstrated that the approach has achieved over 80% of accuracy.
