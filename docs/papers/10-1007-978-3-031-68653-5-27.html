Keywords: Image encryption, Image Decryption, Chebyshev map, Arnold cat map, chaotic maps

In this paper, we introduce a new image encryption algorithm to encrypt gray-scale images. We will use the Arnold Cat map in the confusion part to change the position of each pixel from the original gray-scale image, then we use an external secret key of 160 bits and a double one-dimension (1D) chaotic maps (Chebyshev map), those maps generate two images with identical dimensions as the original gray-scale image, after that, we apply the XOR operation to all generated images. Experiments and security analysis show the value and quality of the proposed approach in terms of accuracy and strength against several attacks.
