IP-Self-Tuning Fuzzy Logic Controller Based on Direct Torque Control to Improve Induction Machine Speed Control

Keywords: Induction machine (IM), Self-tuning, IP Controller (IPC), Fuzzy logic controller, Direct torque control (DTC), Speed drive, Matlab Simulink

The current paper outlines a proposal for an IP (Integral Proportional) self-tuning regulator that uses the fuzzy logic controller by adjusting its proportional gain, to enhance the regulation of the speed of the induction machine through the use of a Direct Torque Control (DTC) strategy. The suggested self-tuning mechanism has the capability to adapt one gain, set against the two gains required for a PI-self-tuning controller. The effectiveness or efficiency of the previously mentioned controller was evaluated against that of a traditional IP controller and a fuzzy PI controller (Proportional Integral) in relation to time response, tracking performance, and disturbance rejection capability. All simulations outcomes ac-quired through the utilization of Simulink and fuzzy tools within MATLAB software for an induction machine with a power rating of 1.5 KW showed excellent performances as well as verified the effectiveness of this controller.
