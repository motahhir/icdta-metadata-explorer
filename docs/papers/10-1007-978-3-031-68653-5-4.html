Keywords: Sentiment Analysis, Neural Network, Backpropagation, Particle Swarm Optimization

Exploring opinions from written or orally expressed data in sentiment classification has become an important decision-making tool that helps companies improve their profits using various machine-learning methods. In this article, we introduce a new method, termed the combined approach, for performing sentiment analysis on consumer opinions it uses a hybrid Artificial Neuron Network ANN algorithm which combines PSO swarm particle optimization and PB backpropagation. The results obtained by the Artificial Neural Network model developed by the PSO-PBB algorithm show very good prediction, better performance, and convergence.
