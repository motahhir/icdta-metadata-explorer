Keywords: Induction Motor (IM), Direct torque Control (DTC), Fuzzy logic Speed Controller (FLSC)

In this paper, the intelligent fuzzy logic approach is used to realize a robust DTC induction machine drive system. Using Mamdani's method, the fuzzy logic speed controller is designed to overcome the limitations of the conventional PI controller. Simulation results have demonstrated the superiority of the proposed controller, it shows better dynamics and disturbance rejection. The fuzzy approach improves the robustness to load disturbances and machine parametric variations as well as the fast response of the DTC controller.
