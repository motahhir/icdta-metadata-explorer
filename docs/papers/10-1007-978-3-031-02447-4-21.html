To improve the existing intrusion detection systems, we propose the use of a set of ML classification algorithms to determine the best one capable of recognizing the maximum number of intrusions in a system. To do this, we used the data set UNSW-NB15 to which we applied data cleaning, then we determined the most important characteristics, and finally we passed to the evaluation of the classifiers with PCA, to apply the different ML algorithms, and conclude that the DT-PCA model gave good results for the classification of the attacks, but which remains to be optimized to preserve more of the characteristics of our data set.
