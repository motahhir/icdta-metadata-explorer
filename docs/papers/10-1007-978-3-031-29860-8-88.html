Robotic Arm Control Using Dynamic Model Linearization and Model Predictive Controller

Keywords: Robotic arm, DOF, Dynamic model, Model Predictive Control, Feedback linearization

Through this work a two Degrees Of Freedom (DOF) articulated robotic arm will be controlled using its dynamic model by a linear model predictive controller. Naturally, a dynamic model of a n-degrees of freedom robotic arm (n ≥ 2) is not linear and each joint torque depends on all joints variables and this makes the joints torques coupled. So the dynamic model must be linearized and its joints torques need to be decoupled. Second, it will possible to apply a linear model predictive control to the linearized version of the dynamic model. To demonstrate the validity of the technique, some simulation results will be given using Matlab/Simulink.
