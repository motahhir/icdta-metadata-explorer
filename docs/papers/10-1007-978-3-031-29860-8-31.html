Keywords: app development, student attendance, android app, learning-management system, database attendance system

Over the years, the manual attendance monitoring process has been the predominant practice. Given the fact that manual processing of attendance can be time-consuming and may produce errors, this has prompted the majority of organizations to use automated attendance monitoring systems to reduce the need for manual attendance tracking. In view of the above, we solve the problem of attendance monitoring via an innovative Android mobile app on smartphone using offline wireless technology, thus reducing the need for Internet connection. The BSSID of the access point can be detected by each student’s smartphone in order to use it as a tracking identifier for attendance. Our application aims to reduce the time taken during the manual process, minimize human errors, and free teachers and administrations from the loads of work often associated with classical attendance checking methods so that efforts can be invested in other more important activities.
