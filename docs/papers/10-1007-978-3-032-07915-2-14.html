Keywords: Blockchain, Data security, Encryption, Decryption, Cloud storage, Hash function

Security is crucial for information systems in the twenty-first century, with the increasing reliance on blockchain and cloud storage highlighting the need for robust security measures to ensure data integrity and confidentiality. (Cryptographic hash algorithms such as SHA-256 and SHA-512 play a key role in this). Data storage safety and reliability are enhanced when blockchain technology is integrated with cloud storage to meet the growing demand for decentralized and scalable data management. (By combining cloud storage with cryptographic hashing techniques, file upload and download processes become both secure and efficient). The goal is to develop a new hashing algorithm that improves the security and performance of integrating blockchain technology with cloud storage, thereby supporting the development of blockchain-based applications. (The proposed blockchain-based cloud storage system demonstrates a clear relationship between processing time and file size, addressing current weaknesses in cryptographic algorithms). This linear connection shows that encryption and decryption times increase in proportion to file size. The project aims to create a hashing algorithm that will enhance both the security and efficiency of blockchain-based cloud storage, resulting in faster encryption and decryption processes, better performance, and greater sensitivity in blockchain applications.
