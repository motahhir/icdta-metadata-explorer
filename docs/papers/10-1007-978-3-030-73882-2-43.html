Keywords: Blockchain, Vehicular ad-hoc networks, Privacy, Security

Vehicular Ad-hoc Network (VANET) is one of the emerging network technologies leading to intelligent transportation systems. Numerous applications are developed for VANETs to provide safety and comfort to drivers. VANET is a special type of Mobile Ad Hoc Network (MANET) in which a temporary network is created using vehicles’ own communication and processing equipment. The data can be exchanged directly from vehicle to vehicle, if there is no communication infrastructure. Consequently, this give rise to various security and privacy threats jeopardizing the availability, confidentiality and integrity of data. Therefore, security and privacy remain the foremost concern in VANETs applications. Recently, blockchain technology is being explored to address security and privacy issues in VANETs. This paper presents the security and privacy requirements for different VANET applications. Moreover, various existing blockchain and non-blockchain based solutions for security and privacy in VANETs are comprehensively reviewed.
