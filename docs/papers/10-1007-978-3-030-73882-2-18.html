Keywords: Brain-computer interface (BCI), Electroencephalogram (EEG), Visual system, Feature extraction, Classification

This work has been carried out in the development of the best BCI systems to recognize EEG visual system signal helps tired drivers, the elderly, patients, and the disabled to drive on good safety condition. The method used in this article based on an application of rhythms (delta, theta and alpha) on EEG acquisition channels to increase the accuracy of the prediction system, after using delta rhythms on selected electrodes using ExtraTrees algorithm, we notice an increase in accuracy from 95.46 to 99.97%, also this system has a good prediction speed of 12560 samples per second during testing. Therefore, instead of minimizing the number of acquisition electrodes, these bad electrodes can be used to increase the performance of the BCI system using rhythms (delta, theta and alpha).
